# Contact Management App

A simple Java-based Contact Management Application that allows users to add, view, search, and delete contacts using a menu-driven console interface.

This project was built to practice core Java programming concepts, including arrays, collections, and user input handling.

## Features

 Add new contacts

 View all saved contacts

 Search contacts by name

 Delete contacts

 Simple interactive console menu

## Tech Stack

Java	
HashMap - collections
Scanner	
Git & GitHub
